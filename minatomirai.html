HIGHLIGHTS

みなとみらいの見どころ

観覧車・展望台・美術館・赤レンガ倉庫…みなとみらいは一日では回りきれないほど見どころが凝縮した横浜随一のエリア。各スポットの詳細は公式サイトも合わせてご確認ください。

01
🎡

コスモクロック21（大観覧車）

人気No.1

高さ112.5mの大観覧車。1周約15分で横浜港・みなとみらい・ベイブリッジを360度眺望できます。夜はLEDイルミネーションで輝き、横浜の夜景に欠かせない存在。カップルや家族連れに特に人気が高く、日没後30分からが最もロマンチックな時間帯です。

💰 大人¥900 / 子供¥500 🕐 営業時間：11:00〜21:00（季節変動あり） 📍 みなとみらい駅 徒歩3分
公式サイトを見る →
02
🏙️

ランドマークタワー スカイガーデン

高さ273mの展望フロア「スカイガーデン」は横浜随一の絶景スポット。晴れた日には富士山・東京スカイツリー・房総半島まで見渡せます。日本最速クラスのエレベーターで地上69階まで約40秒。夜景も非常に美しく、昼夜どちらも楽しめます。

💰 大人¥1,000 / 子供¥500 🕐 10:00〜21:00（月により変動） 📍 みなとみらい駅 直結
公式サイトを見る →
03
🧱

横浜赤レンガ倉庫

明治・大正時代の歴史的建造物を活用した複合商業施設。横浜港を一望できるテラスやイベント広場が人気で、ショップ・レストランが充実しています。年間を通じてクリスマスマーケット・夏のビアガーデンなど季節イベントが開催され、夜のライトアップも見事です。入場は無料。

💰 入場無料（施設利用は別途） 🕐 10:00〜20:00（イベント時は延長） 📍 馬車道駅 / 日本大通り駅 徒歩6分
公式サイトを見る →
04
🎨

横浜美術館

近現代美術を中心に約1万3千点のコレクションを誇る横浜市立美術館。国内外の企画展も年間を通じて充実しており、アート好きには必訪スポットです。みなとみらい駅と地下直結なので雨の日でも快適にアクセスできます。ミュージアムショップのグッズもユニークです。

💰 常設展¥500 / 企画展は別途 🕐 10:00〜18:00（月曜休館） 📍 みなとみらい駅 直結
公式サイトを見る →
05
⛵

横浜港クルーズ

大さん橋・山下公園から出発する観光クルーズ船。ベイブリッジ・みなとみらい・工業地帯を海上から楽しめます。昼のランチクルーズ、夕暮れのサンセットクルーズ、夜景クルーズとシーン別に選べます。陸から見る景色とはまた違う横浜の魅力を発見できます。

💰 ¥1,500〜（コースにより異なる） 🕐 所要約40〜120分（コースにより異なる） 📍 大さん橋 / 山下公園 発着
公式サイトを見る →
06
🛍️

MARK IS みなとみらい

約160店舗が入る横浜最大級のショッピングモール。ファッション・雑貨・レストラン・映画館まで一つの建物で揃います。地下5階・地上6階の大規模施設で、雨の日の観光に最適。ランドマークタワーやクイーンズスクエアと連結しており、一日いても飽きません。

💰 入場無料 🕐 10:00〜21:00（レストランは〜23:00） 📍 みなとみらい駅 直結

NIGHT VIEW

みなとみらいの夜景スポット

横浜の夜景は日本三大夜景のひとつ。みなとみらいだけでも複数の絶景スポットがある。日没から1時間後が最も美しい「ブルーアワー」を狙おう。

🎡
コスモクロック21
観覧車そのものがライトアップ。乗りながら見る横浜夜景は格別。カップル人気No.1スポット。
★★★★★
🏙️
スカイガーデン展望台
273mの高さから見下ろす夜景。横浜港全体・ベイブリッジ・東京方面まで一望できる最上級の夜景。
★★★★★
🌉
大さん橋 屋上
客船ターミナルの屋上は無料の展望スペース。みなとみらい全体を地上から眺める最高のビューポイント。深夜まで開放。
★★★★☆
🧱
赤レンガ倉庫前広場
倉庫とみなとみらいの高層ビル群が同時に見える絶景ポイント。インスタ映えスポットとして超人気。
★★★★☆
🌙 夜景ベストタイム
日没直後（18:00〜19:00頃・季節により変動）から約1時間が「ブルーアワー」と呼ばれる最も美しい時間帯。空が濃紺〜藍色に染まり、ビルの明かりとのコントラストが最高に。その後は完全な夜景に。いずれの時間帯も美しいが、特にブルーアワーは写真映えする。

SEASONAL INFO

みなとみらい季節情報

1年中楽しめるみなとみらいだが、季節ごとに開催されるイベントで雰囲気が大きく変わる。

🌸
春（3〜5月）
最も過ごしやすいシーズン
大岡川の桜並木（3〜4月）、横浜フラワーフェスティバル（5月）。クルーズからの花見も人気。
🌊
夏（6〜8月）
花火・夜景が最高
横浜ナイトフラワーズ（週末花火）、海上花火大会。夕暮れからの夜景クルーズが特に人気。
🍂
秋（9〜11月）
観光ベストシーズン
過ごしやすい気候で観光に最適。赤レンガ倉庫周辺のイベントも多数。空気が澄んで展望台からの眺めも絶好。
🎄
冬（12〜2月）
イルミネーションが最高潮
ヨルノヨ（大規模光のショー）、赤レンガクリスマスマーケット（12月）。冬の澄んだ空気で夜景が最も美しい時期。
